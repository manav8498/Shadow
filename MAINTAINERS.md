## How maintainership works

See [GOVERNANCE.md](./GOVERNANCE.md). In short: sustained high-quality
contribution over ≥3 months earns an invitation from an existing
maintainer. There is no self-nomination.

## Path-based review routing

Per-directory reviewer routing is enforced by
[`.github/CODEOWNERS`](./.github/CODEOWNERS). Update that file in the
same PR whenever maintainership changes.
